How to Add

  1. Copy the one you’d like to clipboard
  2. Open Settings in Gmail (gear icon on top right) > See all settings
  3. + Create New (down in the Signatures section of the General settings page)
  4. Paste
  5. Add signature defaults and Save

Tips

We recommend pasting your signature in Safari or Firefox for the best result. In Chrome, links in your signature will get the default underline style—this won’t appear right after you paste, but after you refresh the Gmail settings page or start using the signature. If the underline doesn’t bother you, Chrome is fine; for a cleaner look, use Safari or Firefox.
